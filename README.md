# Holdings Table

This is a single-page React application that displays a holdings table. The table data is fetched from an API and grouped based on asset class.

## Hosted Link

The application is hosted on Netlify. You can view it [here](https://neerajcode34-hirequotiont.netlify.app/).

## Features

- Displays holdings table with the following columns:
  - Name
  - Ticker
  - Asset Class
  - Average Price
  - Market Price
  - Latest Change (%)
  - Market Value (Base CCY)
- Holdings can be expanded or collapsed based on asset class.
- Alternating background colors for table rows.
- Text alignment centered in the table cells.
- Holding names and asset class names are bold.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/neerajcode34/HireQuotient-Assignment
```

2. Navigate to the project directory:

```bash
cd HireQuotient-Assignment
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm start
```

5. Open your browser and go to [http://localhost:3000](http://localhost:3000) to view the app.
